# Sales Analytics Dashboard - Power BI

Interactive business intelligence dashboard built in Power BI for sales performance analysis and KPI tracking.

## Tech Stack
- Power BI
- DAX
- Power Query
- Data modeling

## Features
- KPI cards (Revenue, Profit, Orders)
- Sales trends over time
- Regional performance analysis
- Product/category breakdown
- Interactive filters and slicers

## Process
- Data cleaning and transformation (Power Query)
- Data modeling with relationships
- DAX measures for KPIs and calculations
- Interactive dashboard design

## Outcome
Provides clear business insights and supports data-driven decision-making.
